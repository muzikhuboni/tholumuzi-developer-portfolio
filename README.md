### Setup

1. Clone the repository
```bash
git clone git@github.com:sanjibroy360/portfolio-sanjib.git
cd portfolio-sanjib
```

2. Install Packages
```bash
yarn install
```

3.  Then, you can run it locally in development mode
```bash
yarn dev
```
